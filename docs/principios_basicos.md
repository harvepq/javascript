# Tipos de datos

## 10 tipos de datos

|Primitivos|Complejos|
|--------------|-------------|
|string        |object       |
|number        |array        |
|boolean       |function     |
|null|
|undefined|
|symbol|
|bigint|

Los datos Primitivos son ***inmutables*** es decir se pasan por valor</br>
Los datos Complejos son ***mutables*** es decir se pasan por referencia</br>
